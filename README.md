# Speed Test

Aplicativo desktop em Python que mede a velocidade real de download e upload da sua conexão de internet, usando a infraestrutura pública de testes de velocidade.

## Sobre o projeto

Este projeto começou como um app desktop feito com Tkinter. Está evoluindo, fase por fase, para uma versão web full-stack, com uma API própria e um frontend em HTML/CSS/JS.

## Tecnologias

- Python 3
- Tkinter — interface gráfica
- Pillow — carregamento e redimensionamento dos ícones
- speedtest-cli — medição real de velocidade de download e upload

## Como rodar

```bash
git clone https://github.com/willvinic/Internet-speed-test.git
cd Internet-speed-test
pip install -r requirements.txt
python main.py
```
